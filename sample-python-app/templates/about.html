Project Overview

This is a demonstration Python Flask web application that showcases a complete DevOps pipeline for deploying applications to Azure VMs. The project demonstrates Infrastructure as Code (IaC) principles using Terraform and automated deployment using Azure DevOps Pipelines.

New in v2.0.0: Enhanced deployment tracking, new API endpoints, and improved monitoring capabilities!
Technology Stack
  • Python 3.x
  • Flask Web Framework
  • Gunicorn WSGI Server
  • Nginx Reverse Proxy
  • Bootstrap UI
Infrastructure
  • Microsoft Azure
  • Azure Virtual Machines
  • Terraform IaC
  • GitHub Actions CI/CD
  • Ubuntu Linux

Architecture

The application follows modern cloud-native principles:

  1. Infrastructure Provisioning: Terraform creates Azure resources (VM, networking, security groups)
  2. Automated Setup: Custom script extension installs and configures the environment
  3. Application Deployment: GitHub repository is cloned and Flask app is deployed
  4. Service Management: Supervisor and systemd manage application lifecycle
  5. Web Server: Nginx acts as reverse proxy with load balancing
  6. Monitoring: Health checks and system monitoring endpoints

Security Features

  • SSH Key Authentication
  • Password Authentication Disabled
  • UFW Firewall Configuration
  • Network Security Groups
  • Non-root Service Execution
  • Secure Service Management

Source Code

This is an open-source demonstration project. The complete source code, including Terraform configurations and CI/CD pipeline definitions, is available for learning and reference.
